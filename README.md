# Agro-Nexus 🌾
**Agro-Nexus: An AI-Driven Digital Ecosystem for Crop Disease Detection and Mandi Analytics**

*(Also documented as: Agro-Nexus: AI-Powered Crop Disease Detection and Mandi Analytics System for Academic Records)*

Agro-Nexus is a cutting-edge solution designed to transform traditional farming using Artificial Intelligence. This platform empowers farmers by providing instant disease diagnosis and real-time market insights to ensure better crop management and profitability.

## ✨ Our Features
* **🔬 AI Disease Detection:** Powered by advanced AI models for high-accuracy crop diagnosis.
* **📊 Mandi Analytics:** Real-time price tracking and historical trend analysis for agricultural commodities via OGD APIs.
* **☁️ Weather Insights:** Hyper-local weather forecasting to assist in critical farming decisions.
* **📱 PWA Experience:** Optimized as a Progressive Web App for a seamless, full-screen mobile experience.

## 🛠️ Technology Stack
* **Frontend:** React.js, Next.js (Deployed on Vercel)
* **Backend:** Python (Flask), Flask-CORS
* **AI Integration:** Multi-modal analysis for crop health monitoring.
* **Data Management:** Firebase (Authentication, Firestore, Hosting)
* **API Integration:** Requests, OpenWeather API, Custom Mandi Price Scrapers
